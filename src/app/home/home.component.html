This is a simple tracking application

How it works ?

to start tracking just click on play_circle_filled so you can remark that the time was started.

you can also reset the timer with restore or pause and resume with pause_circle_filled

finally, you want to finish tracking, add description on the description field, and press on stop

these information will posted to the server and added to your list bellow
